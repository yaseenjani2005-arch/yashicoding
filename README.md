# Verification-first reduced-model cold-plate screening

Public source-and-summary mirror for a bounded computational thermal-screening study.

## Evidence boundary

This mirror contains text source, protocols and compact result summaries. It does **not** claim a matched D6 experiment or an executed independent 3-D conjugate CFD campaign. The 3-D case deck is frozen and execution-ready, but all cases remain `NOT_EXECUTED` until meshes, solver logs, fields and uncertainty audits are deposited.

Large checkpoints, POD objects, field arrays and casewise output archives are not duplicated through the GitHub contents interface. They remain identified by SHA-256 in the full archival package and should be deposited in Zenodo/OSF before journal submission.

- Retrospective locked non-case-specific model-form holdout.
- Frozen split-conformal calibration and a third untouched 3,600-case guard campaign.
