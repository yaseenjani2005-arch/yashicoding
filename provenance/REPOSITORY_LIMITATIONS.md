# Repository limitations

This GitHub mirror is not yet a complete archival release. It omits large binary checkpoints, POD objects, field arrays and full casewise outputs because the available connector does not provide a suitable release/LFS workflow. The full local package contains these artifacts and SHA-256 manifests.

Before submission, archive the complete ZIP in Zenodo, OSF, Figshare or an institutional repository and add the DOI to `CITATION.cff` and the manuscript data-availability statement.

The independent 3-D CFD campaign is not executed. The repository must not be tagged as a validated 3-D release until raw meshes, solver settings, residual histories, GCI/time-step analyses and fields are included.
